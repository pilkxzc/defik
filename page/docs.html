Документація Yamato

Yamato — це сучасна торгова платформа для криптовалют, що об'єднує в собі моніторинг ринку в реальному часі, автоматизовану торгівлю через ботів та зручне управління портфелем.

Платформа створена для трейдерів будь-якого рівня — від початківців до професіоналів. Основні можливості:

  • Графіки в реальному часі з інтеграцією TradingView
  • Автоматичні торгові боти з підключенням до Binance
  • Демо-режим для безпечного навчання без ризику
  • Управління портфелем з детальною аналітикою
  • Лімітні та ринкові ордери
  • Двофакторна автентифікація та Passkeys для безпеки

Початок роботи

Щоб почати користуватись Yamato, виконайте кілька простих кроків:

Реєстрація

1
Перейдіть на сторінку /register та заповніть форму: ім'я, email, пароль.
2
Підтвердіть email — на вашу пошту прийде лист з посиланням для верифікації.
3
Після підтвердження увійдіть у свій акаунт на сторінці /login.

Демо та реальний рахунок

Yamato пропонує два типи рахунків:

  • Демо-рахунок — віртуальні кошти для тренування. Ви можете торгувати без ризику та вивчати платформу.
  • Реальний рахунок — для торгівлі реальними криптовалютами через підключення до Binance.

Переключення між рахунками відбувається у профілі або на дашборді.

Рекомендуємо спочатку попрацювати на демо-рахунку, щоб освоїтись з інтерфейсом та протестувати стратегії.

Дашборд

Дашборд — це головна робоча панель трейдера. Тут зібрано все необхідне для моніторингу ринку та прийняття торгових рішень.

TradingView графік

Інтерактивний графік з підтримкою всіх основних індикаторів та інструментів технічного аналізу. Дані оновлюються в реальному часі безпосередньо від Binance.

  • Зміна таймфрейму (1m, 5m, 15m, 1h, 4h, 1d тощо)
  • Додавання індикаторів (MA, RSI, MACD та інші)
  • Інструменти малювання (лінії тренду, рівні)
  • Перемикання між торговими парами

Віджети та кастомізація

Дашборд складається з віджетів, які можна налаштувати під свої потреби:

  • Перетягування (drag) — змінюйте порядок віджетів перетягуванням
  • Зміна розміру (resize) — налаштуйте розмір кожного віджета
  • Налаштування зберігаються автоматично

Ринкові дані

На дашборді відображаються актуальні ціни криптовалют, зміна за 24 години, обсяг торгів та інші ключові метрики.

Портфель

Сторінка портфелю надає повний огляд ваших активів та фінансового стану на платформі.

  • Загальний баланс — сума всіх ваших активів у доларовому еквіваленті
  • Розподіл активів — графічна візуалізація структури портфелю
  • Список холдингів — детальна інформація по кожному активу: кількість, поточна ціна, прибуток/збиток
  • Історія транзакцій — повний лог всіх операцій: покупки, продажі, перекази
Регулярно перевіряйте портфель для контролю диверсифікації та оцінки ефективності ваших торгових стратегій.

Торгові боти

Торгові боти — потужний інструмент автоматизації торгівлі. Бот виконує угоди за заданими параметрами 24/7 без вашої участі.

Створення бота

1
Перейдіть на сторінку /bots та натисніть кнопку створення нового бота.
2
Оберіть тип бота, торгову пару та налаштуйте параметри стратегії.
3
Для реальної торгівлі вкажіть API ключ Binance (API Key та Secret). Ключ повинен мати дозвіл на торгівлю.
4
Вкажіть суму інвестиції та активуйте бота.
Ніколи не передавайте API ключі третім особам. Використовуйте ключі з обмеженими дозволами (тільки торгівля, без виведення коштів).

Режими роботи

  • Paper (паперова торгівля) — бот імітує угоди без реальних операцій. Ідеально для тестування стратегій.
  • Live (жива торгівля) — бот виконує реальні операції через Binance API.

Управління ботами

  • Моніторинг — відстежуйте прибуток, кількість угод та стан бота в реальному часі
  • Активація/Пауза — запускайте або зупиняйте бота в будь-який момент
  • Дублювання — створіть копію бота з тими самими налаштуваннями
  • Статистика — детальні звіти по кожному боту на сторінці /bot-stats/:id
  • Видалення — видаліть бота, якщо він більше не потрібен

Налаштування відображення

Кожен бот має налаштування відображення (display settings), які дозволяють змінити вигляд інформації на панелі бота: які дані показувати, формат відображення тощо.

Ордери

Yamato підтримує різні типи ордерів для гнучкої торгівлі:

Типи ордерів

  • Ринковий ордер (Market) — миттєва покупка або продаж за поточною ринковою ціною.
  • Лімітний ордер (Limit) — ордер виконається, коли ціна досягне вказаного рівня. Дозволяє купити дешевше або продати дорожче.
  • Стоп-лосс (Stop Loss) — автоматичний продаж активу при досягненні мінімальної ціни для обмеження збитків.
  • Тейк-профіт (Take Profit) — автоматичний продаж при досягненні цільової ціни для фіксації прибутку.
Використовуйте комбінацію стоп-лосс та тейк-профіт для кожної позиції — це захистить ваш капітал та зафіксує прибуток.

Профіль та безпека

Управління акаунтом та налаштування безпеки знаходяться на сторінці /profile.

Редагування профілю

  • Зміна імені та аватара
  • Оновлення email адреси
  • Зміна паролю

Двофакторна автентифікація (2FA)

Для додаткового захисту акаунту рекомендуємо увімкнути 2FA:

1
Перейдіть у налаштування безпеки в профілі.
2
Відскануйте QR-код додатком-автентифікатором (Google Authenticator, Authy тощо).
3
Введіть 6-значний код для підтвердження та активації 2FA.

Passkeys

Yamato підтримує сучасний стандарт WebAuthn (Passkeys) — вхід за допомогою біометрії або апаратного ключа безпеки. Це найбезпечніший спосіб автентифікації.

Прив'язка Telegram

Прив'яжіть свій Telegram-акаунт для отримання сповіщень про важливі події: спрацювання ботів, виконання ордерів, зміни на ринку.

Обов'язково увімкніть хоча б один метод додаткового захисту (2FA або Passkey) для безпеки вашого акаунту.

Підписки

Yamato пропонує різні плани підписки з розширеним функціоналом:

  • Безкоштовний план — базовий доступ до платформи, обмежена кількість ботів та функцій.
  • Платні плани — розширені можливості: більше ботів, додаткові індикатори, пріоритетна підтримка та ексклюзивні функції.

Перегляньте доступні плани та їх можливості на сторінці /subscriptions.

Новини

Стрічка новин (/news) містить актуальну інформацію про крипто-ринок:

  • Аналіз ринку та прогнози
  • Важливі події в світі криптовалют
  • Оновлення платформи Yamato
  • Навчальні матеріали

Новини можна фільтрувати за категоріями для швидкого пошуку потрібної інформації.

Спільнота

Приєднуйтесь до спільноти Yamato в соціальних мережах:

  • Telegram — торгові сигнали, аналіз ринку та оновлення в реальному часі
  • Instagram — контент про трейдинг та успішні угоди
  • YouTube — навчальні відео, огляди ринку та гайди
  • TikTok — короткі відео та швидкі лайфхаки

Всі посилання доступні на сторінці /community.

Часті питання (FAQ)

Чи безпечно зберігати API ключі Binance на платформі?
Так, API ключі зберігаються у зашифрованому вигляді. Рекомендуємо створювати ключі з обмеженими дозволами — тільки для торгівлі, без можливості виведення коштів. Також радимо обмежити доступ ключа за IP-адресою у налаштуваннях Binance.
Що таке Paper-торгівля?
Paper-торгівля (паперова торгівля) — це режим симуляції, де бот виконує угоди на віртуальних коштах. Це дозволяє протестувати стратегію без фінансового ризику. Результати максимально наближені до реальних ринкових умов.
Як отримати демо-кошти?
Демо-кошти нараховуються автоматично при реєстрації. Якщо ваш демо-баланс закінчився, ви можете скористатись функцією поповнення (faucet) для отримання додаткових віртуальних коштів.
Як підключити Telegram-сповіщення?
Перейдіть у налаштування профілю та знайдіть секцію Telegram. Натисніть кнопку прив'язки та слідуйте інструкціям Telegram-бота. Після підключення ви будете отримувати сповіщення про торгові операції та важливі події.
Чи можу я мати кілька ботів одночасно?
Так, кількість одночасно активних ботів залежить від вашого плану підписки. На безкоштовному плані є обмеження, а платні плани дозволяють запускати більше ботів паралельно.
Що робити, якщо я забув пароль?
На сторінці входу натисніть "Забули пароль?" та введіть свій email. Вам прийде лист з посиланням для скидання паролю. Посилання активне протягом обмеженого часу.
Які криптовалюти підтримує платформа?
Yamato підтримує всі торгові пари, доступні на Binance. Це включає основні криптовалюти (BTC, ETH, BNB та інші), а також сотні альткоїнів. Дані оновлюються в реальному часі.

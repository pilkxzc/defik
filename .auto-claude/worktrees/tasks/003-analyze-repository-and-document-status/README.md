# Yamato Trading Platform

## Огляд проекту

Yamato — це веб-платформа для торгівлі криптовалютами з підтримкою демо-рахунків, торгових ботів, портфельного відстеження та адмін-панелі. Побудована на Express.js (бекенд) з SQLite базою даних, Socket.IO для реального часу та ванільним JS фронтендом.

## 🔍 Статус проекту

> Останній аудит: 2026-03-02

### Компоненти системи

| Компонент | Статус | Деталі |
